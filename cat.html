¿Qué entendí que era o que hace una API?

Una API es como una herramienta que la pidemos utilizar como un puente para mandar una solicitud de algo que nuestro proyecto o aplicacion necesita, un ejemplo puede ser de que una app de mapas, lo que hace es mandar a las apis una solicitud para obtener mapas y no hacerlos desde cero y no estresarse :3.
